# Responsive Resume

This project contains code for a responsive resume built using HTML, CSS, and JavaScript.

## Files Included

- **index.html** - The main HTML file containing the resume structure and content.
- **style.css** - The CSS file for styling and making the resume responsive on different devices.
- **script.js** - JavaScript file that adds interactivity, such as toggling job details in the experience section.

## Features

- Responsive layout that adapts to desktop, tablet, and mobile screen sizes.
- Toggle buttons in the experience section allow showing or hiding detailed job descriptions.
- Clean, modern design with easy-to-read typography.
- Links to GitHub and LinkedIn profiles included in the footer.

## How to Use

1. Open `index.html` in any modern web browser.
2. Resize the browser window or open on different devices to see the responsive behavior.
3. Click "Show Details" buttons in the Experience section to expand or collapse job details.

## Customization

- Update the personal information, skills, experience, and education sections in `index.html` as needed.
- Modify the styles in `style.css` to change colors, fonts, or layout.
- Extend the functionality in `script.js` if you want to add more interactivity.
